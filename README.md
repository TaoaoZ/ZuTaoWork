# 企业经营管理驾驶舱 H5 原型

入口：`index.html`

页面内交互：

- 底部导航：总览 / 排行 / 督办
- 排行 Tab：综合 / 经营 / 运营 / 内控 / 舆情 / 创新 / 安全
- 总览「企业经营六大指标」：点击任一指标卡片打开对应健康详情

本地运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

`start.ps1` 会启动本地静态服务器并打开 `http://127.0.0.1:4173/index.html`。关闭启动脚本窗口即可停止服务器。

说明：综合、经营、运营、总览、督办和弹窗视觉基线来自 MasterGo；内控、舆情、创新、安全排行按照同一套企业驾驶舱设计语言补齐。
